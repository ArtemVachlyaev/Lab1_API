# Лабораторная работа №1

**Тема:** Создание простого Web API на ASP.NET Core, вызов из консольного приложения и тестирование через Postman

## Цель работы

Научиться:
- создавать простое Web API на C#
- обрабатывать HTTP-запросы (POST)
- использовать модель для передачи данных (Model Binding)
- вызывать API из консольного приложения с помощью `HttpClient`
- тестировать API через Postman

## Вариант задания

**2-умножение** 

## Технологии

- ASP.NET Core Web API 
- C#
- Postman
